Homework Assignment by Jason Fetzer

Assignment 3, Bing Maps QuadKey Demo

  • These assignments were prepared for GeoSpatial Data and Visualization at Illinois Institute of Technology, Chicago, IL. The Bing Maps Documentation for this assignment can be found here

    For Google Maps, visit This link

  • Code Sample from C# (for Tile System) was ported to Python & Javascript and used in Django MVC framework. The tilePython.py file consists of the Tile System class. The application is deployed on Python Anywhere, and documentation can be found here. The working functions are a product of Javascript, not the Python implementations or C#.

  • Documentation for Javascript implementations can be found through the Here Developer (Traffic) API (the docs are better than Bing!) and further reading of the Google Roads API

  • The ideas of the original HW consist of being able to click both points on the map, pass them to input boxes, calculate resuls, and link to the tile page. The "alerts" show the results of the calculations from user input. This will serve as a base for the final application.

  • The final application is meant to demonstrate how automated map matching can assist in Development Applications from sliced data.
